                SELECT mov_title AS "Movie Title", mov_time AS "Runtime", mov_rel_country AS "Release Location"

                FROM movie

                WHERE mov_id =
 
                       (SELECT mov_id

                       FROM rating 

                       WHERE rev_stars > 10);

The above SQL query does the following:
- Firstly, it selects the movie title, the movie runtime and the release country and formats the coloumns so they are more readable.
- Secondly it then selects the above columns from the movies table and filters these from the rest of the columns.
- Lastly the subquery returns the movies table but filtered using the ratings table, returning how highlighy rated they are and esnuring it okay only returns well rated films.
